# 🌾 KisanDost: The AI-Powered Digital Companion for Farmers

> **Developing the future of agriculture with precision technology and community-driven insights.**

KisanDost is a comprehensive digital platform designed to empower Indian farmers with cutting-edge AI tools, real-time data, and localized knowledge. Built for the modern agricultural landscape, it bridges the gap between traditional wisdom and technological innovation.

---

## 🚀 Key AI Features (Judge's Highlights)

### 1. 💰 AI Profit Predictor
The Profit Predictor enables farmers to make data-backed financial decisions before they even sow a seed.
- **Precision Inputs**: Factors in crop type, soil type, and regional market prices.
- **Financial Forecasting**: Generates detailed reports on expected net profit in INR (₹).
- **AI Smart Insights**: Provides automated recommendations on optimization, price trends, and confidence scores (85%+ accuracy baseline).
- **Visual Analytics**: Interactive charts powered by modern UI components to visualize expenditure vs. revenue.

### 2. 📈 Yield AI (Yield Predictor)
A geolocated tool that uses environment-specific data to forecast harvest output.
- **Interactive Farm Mapping**: Integrated with Leaflet.js to pinpoint farm locations via coordinates.
- **Weather-Sync Technology**: Automatically syncs current localized weather data (temperature, humidity, atmospheric pressure) into the prediction model.
- **Benchmarking**: Compares predicted yield (tons/hectare) against regional averages to help farmers identify if they are over or under-performing.

### 3. 🛡️ AI-Enhanced Crop Care (Disease Detection)
A robust diagnostic tool for maintaining crop health and preventing loss.
- **Interactive Diagnosis**: Farmers can select from major Indian crops (Wheat, Rice, Maize, Cotton, Tomato, Groundnut, Mango).
- **Localized Wisdom**: Full symptoms, favorable conditions for disease spread, and management protocols available in **English, Hindi, and Gujarati**.
- **Control Measures**: Precise chemical and biological control recommendations (e.g., Propiconazole dosage, Sulfur-based applications).

## 🛠️ Technology Stack & Innovation

KisanDost is built using a modern, high-performance stack to ensure accessibility and speed:

- **Framework**: **Next.js 16 (Turbopack)** for blistering fast page loads and seamless routing.
- **Language**: **TypeScript** for robust, type-safe business logic.
- **Styling**: Custom **Tailwind CSS** design system featuring a premium, emerald-themed aesthetic (glassmorphism/dynamic animations).
- **Internationalization**: Fully localized via **Next-Intl** (English, Hindi, Gujarati) to reach farmers in their native tongue.
- **Authentication**: Secure login and profile management via **Clerk**.
- **Real-time Data**: Integrated **Weather API** for village-level accuracy.

## 🗺️ Other Features

## 🏁 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Access the Portal**:
   Open [http://localhost:3001](http://localhost:3001) in your browser.
